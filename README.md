# 后台管理系统
## “妆瓣”项目后台管理系统1.0版，使用Vue搭建，UI组件为iView,前后端数据传输使用Axios，后端服务已部署在服务器

# 项目运行说明：
###  1.解压缩包
###  2.cd到该项目
###  3.npm install   安装以来包 (前提电脑已安装node.js, 全卷安装vue ,vue-cli脚手架,未安装则运行失败)
###  4.浏览器输入：http://localhost:8080/?#/
###  5.输入用户名与密码登陆系统（未设置，随便输入）
